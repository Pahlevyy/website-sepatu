 Sepatu Store NIKU
Website toko sepatu sederhana dengan frontend statis dan backend Node.js.
Project ini dibuat sebagai latihan dan portofolio (CV).

 Tech Stack
- HTML, CSS, Bootstrap
- JavaScript (Fetch API)
- Node.js
- Express.js
- JSON file (simulasi database)

 Features
- Menampilkan testimoni dari backend
- Form pemesanan sepatu
- Data pesanan tersimpan ke file JSON
- REST API menggunakan Express
